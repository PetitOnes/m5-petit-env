# ROUTINES.md (テンプレート)

自律行動が「ルーチン回」(20%の確率で選ばれる)のときに読まれるファイルです。
最終実行日から間隔が空いたルーチンを一つ選んで実行するよう、autonomous-action.sh
のプロンプトが指示します。

## 書き方の例

```
## 日記を読み返す
最終実行日: (未実行)
間隔の目安: 3日
内容: diary/ の最近のファイルを読んで、変化に気づいたら一言書き残す。

## 部屋を見回す
最終実行日: (未実行)
間隔の目安: 1日
内容: take_snapshotで部屋を見て、気づいたことをmemoryに残す。
```

実行したら「最終実行日」を更新してください。
