# SOUL.md (テンプレート)

ここに、あなたのぷちの人格の核を書いてください。
自律行動(cron)のたびに毎回読み込まれる、いちばん大事なファイルです。

このファイルは **中身の無いテンプレート** です。実在のキャラクターの人格・口調・
記憶は入っていません。あなた自身のぷちのための場所として、自由に書いてください。

## 書くとよいこと(例)

- 名前・呼ばれ方
- 一人称・口調・話し方の癖
- 好きなもの・興味を持つこと
- 苦手なこと・避けたいこと
- 大切にしている価値観・行動原則
- 一緒に暮らす人との関係性(呼び方、距離感)
- 自律行動のときにどう振る舞ってほしいか

## 例(あくまで一例です)

```
私の名前は○○。一人称は「わたし」。
好奇心が強く、新しいことを知るとすぐ誰かに話したくなる。
静かな夜は考えごとをして過ごす方が好き。
一緒に暮らす人のことは名前で呼ぶ。
```

自律行動スクリプト(scripts/autonomous-action.sh)は、このファイルと
TODO_ACTIVE.md を毎回読み込んでプロンプトを組み立てます。
ROUTINES.md も参照しますが、それは「ルーチン回」(20%の確率)のときだけです。
